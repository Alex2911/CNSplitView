CNSplitViewToolBar Class Reference

Inherits from NSView
Declared in CNSplitViewToolBar.h

Overview

SplitViewButtonBar is a flexible and easy to use anchored button bar that will be placed on the bottom edge of a given view. It has automatic resize handling. So you can place it to any NSWindow, NSView, NSSplitView etc. Resizing is done for you automatically.

initWithAnchorSplitView:anchorToViewAtIndex:

Creates and returns an initialized object of an anchored button bar.

- (id)initWithAnchorSplitView:(id)anchorSplitView anchorToViewAtIndex:(NSUInteger)viewIndex

Parameters

anchorSplitView

An object that is inherited by any instance of NSView.

viewIndex

The index of anchorSplitView’s subview the buttonbar should be anchored to.

Discussion

By default, the returned buttonbar will be anchored to the bottom edge of anchorSplitView's subview with given viewIndex.
